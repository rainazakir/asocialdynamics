# Bio-inspired decision making in swarms under biases from stubborn robots, corrupted communication, and independent discovery

Repo contains the code to run Gillespie simulation, the robot simulation code and the code to process Gillespie and robot simulation data to generate the figures.

### Running Robot simulations
ArgosCode_antagonistic contains the robot controller to run experiments with antagonistic asocial dynamics and ArgosCode_synergetic contains code to run experiments with synergwtic bias.
The robot controller can be found at `ARGoS_simulation/behaviours/agent_stub.c`. 

You need to have argos3 and Kilobot plugin for argos3 to run the simulations.
